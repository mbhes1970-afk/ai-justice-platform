## Installatie

```bash
npm install
cp .env.example .env
# Vul je MISTRAL_API_KEY in
npm start
```

## Netlify Deploy

1. Push naar GitHub
2. Verbind repo met Netlify
3. Stel `MISTRAL_API_KEY` in als environment variable
4. Deploy
